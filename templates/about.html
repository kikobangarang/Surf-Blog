🌊 About This Site 🌊

Hey there! Welcome to my surf blog experiment, a project I built while learning Django. This site is all about surfing, but more importantly, it's where I tested out web development concepts like templates, views, models, and web scraping.

🏄 What This Site Does

  • Displays surf-related Blog Posts
  • Uses web scraping to pull surf forecasts
  • Lets me practice working with Django, Python, and HTML/CSS

🌊 Why I Built It

I wanted to get hands-on experience with Django and explore how to integrate web scraping into a project.

⚙️ Tech Stack

  • Backend: Django 🐍
  • Frontend: HTML, CSS (Bootstrap) 🎨
  • DataBase: MySQL for blog posts 🗃
  • Data: Web scraping for surf forecasts 🌊

Building this project has been an invaluable experience in mastering Django fundamentals, from setting up views and templates to implementing web scraping. It has strengthened my understanding of full-stack development and how to integrate dynamic data into a web application.
